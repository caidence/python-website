Really just Shorthand

Assignment operators are simple ways to do math on a variable, and reassign its value at the same time. Here is an example of how you would do this without assignment operators. Say you have a variable x which is equal to five, and you want to set the value of x equal to itself plus five. To do this, you could write x = x + 5. This statement takes the variable x and assigns it a new value of itself plus five. Variable reassignment is very common in programming, so common that there is a shorter way to do it. The expression x += 5 is exactly the same as x = x + 5. += is called an assignment operator. There is an assignment operator for each math operator that we learned about in the math section. Another example, say we have that same variable, x = 5. We can also subtract and reassign this variable using -=. That would look something like x -= 3. After this expression, the new value of x will be 2. You can also use *=, /=, **=, %=, and //=. Literally all this expression does, is change the value of the variable to itself after the math operator has performed its operation on the original variable.

>>> a = 5
>>> d = 10
>>> c = 7
>>> d = 3
>>> print(a += 5)
10
>>> print(b -= 3)
7
>>> print(c *= 10)
70
>>> print(d /= 1.5)
2

In the first example, we print a += 5. Because the value of a is five and a += 5 is just short for a = a + 5. We get back the value 10. The next few examples demonstrate that you can use this shorthand assignment operator for any basic math operator. This is a very useful tool to have. Programmers find this so handy, that Python is not even the only language that uses it. Nor is Python the language that invented its syntax. It is commonly used in languages like Java, PowerShell, C, and C++, too!
