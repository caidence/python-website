keys, values, items

>>> test = {'test1': 1, 'test2': 2}
>>> test['test1']
1
>>> test['test2']
2

Dictionaries are collections of key value pairs. Generally, you use the key of a dictionary to retrieve the value stored at that key. dictionary['key'] will return the value stored at a key in the dictionary. In the example, test['test1'] returns 1 because that is the value stored at the 'test1' key. In a dictionary, we associate a key with a value by placing a colon between the key and its value, key: value. We can separate key value pairs by placing a comma between them. Make sure that you create your dictionary with curly braces { } otherwise, it will not be a dictionary, and you may get an error.

As you learned in the data types tutorial, dictionaries have key value pairs. The main use case of a dictionary is to use the key to retrieve the value. These methods are common for getting the keys, values, or both! Starting off, keys will return a view object of the dictionary's keys. Likewise, values will return a view object of all of the dictionary's values. items will return a view object of two item tuples containing the dictionary's key value pairs. You can treat the returned view objects from these method calls as you would treat a list or tuple.

>>> person = {"Name": "Joe", "Age": 30, "Hobby": "Programming"}
>>> # Get the dict's keys
>>> person.keys()
dict_keys(['Name', 'Age', 'Hobby'])
>>> # Get dict's values
>>> person.values()
dict_values(['Joe', 30, 'Programming'])
>>> # Get dist's key value pairs
>>> person.items()
dict_items([('Name', 'Joe'), ('Age', 30), ('Hobby', 'Programming')])

There is a lot going on here so let's break it down. On the first line, we have a dictionary named person. After the dictionary declaration, we get a view object of just the keys by calling the keys method on the person object. Don't be intimidated by the return value of these method calls, you can treat the returned object exactly like a list or tuple. Going on, we then get a view object of the values by calling the values method on the person dictionary. Again, the return object acts like a list or tuple. Finally, we call the items method on the person object. This returns a list of tuples containing the dictionary's key value pairs. These methods will make more sense and appear more useful when you learn about for loops.

update, del

The next two topics are update and del. del is not a method, but I thought it appropriate to introduce it here, considering that – as of this writing – there is no remove method for dictionaries. update is used to add key value pairs to an existing dictionary and/or change the value of an existing key. You can use update to add or update more than one key value pair to a dictionary at a time. You can use del to remove key value pairs from a dictionary. The syntax of del is different than what you're used to, but you will get used to it over time.

>>> example = {'one': 1, 'two': 2}
>>> example.update({'three': 3, 'one': 'uno'})
>>> example
{'one': 'uno', 'three': 3, 'two': 2}

There are a few things to notice here. Do you see how when we pass an object to the update method, the object that is passed is a dictionary? Also notice, we are updating an existing value, and adding a new value at the same time. If a key already exists when you call the update method, Python will override the existing key in the dictionary with the new one. The next thing to notice is in the output. Can you spot it? Dictionaries are an unordered collection of key value pairs. There does exist an object in Python called an OrderedDict, if you want to check it out, it lives in a module called collections. In this example specifically, a key named one already exists in the dictionary, so when we call update, passing a key value pair with a key being 'one', the 'one' key is overridden in the existing dictionary with the value 'uno'.

Like I said earlier, the del keyword is not a method, nor is it specific to dictionaries. However, we can use it with dictionaries to remove a key value pair, by key. Say we wanted to remove the key value pair {'delete_me': -1} from the dictionary example. We would write del example['delete_me'].

>>> test = {'to-delete': 1, 'to-keep': 2}
>>> del test['to-delete']
>>> test
{'to-keep': 2}

When we delete a key value pair from a dictionary, we only need to use the key with the delete keyword. Also, notice that we need to have the dictionary's name before the square braces with the key name of the key value pair to delete. You can think of deleting a key value pair much the same that you would access the value stored at a key, just prepend the statement with the del keyword.
