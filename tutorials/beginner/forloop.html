For loop

The for loop is a very basic iterator in Python. It is used to step through items in ordered sequences such as lists, tuples, and strings. The header has the general format: for item in object:. This will execute a block of code for every item in the iterable object. In the previousexample of a for loop, header, item used to refer to the current item in the object that the for loop is executing the block of code for.

>>> example = [1, 2, 3, 4]
>>> total = 0
>>> for item in example:
...     total += item
>>> total
10

First, we create a list with a few values in it. The objective of this program is to get the sum of the list. Next, we create a variable, total, which we update with every item in the for loop to keep track of the running total sum of the items in example. After that, we have the for loop statement. This line is saying "for every item in the example list, do the code indented below." Since a list is an ordered collection in Python, we can use it with a for loop. In the example, item is used to refer to the current item the for loop is executing code with/on. The for loop will start by executing the indented code on the item at index 0 in the object example. After that, the loop will execute the indented code on index 1, and so on until the loop has executed the code on every item in the list. In the case of this example, all we do is add the item to the variable total. After the loop, when we view the contents of total, we can see that it is equal to the sum of the numbers in example.

>>> sentence = ('Hello', "I'm", 'Bill')
>>> for word in sentence:
...     print(word, end=" ")
Hello I'm Bill

This program takes a tuple of words and prints them sequentially with a space between each word. Notice how this time we used word to reference the current object in the sequence instead of item. This is just a variable, and its name is arbitrary to the execution of the program.

>>> test = [(1, 2, 3), (4, 5, 6)]
>>> for (a, b, c) in test:
...     print(a + b + c, end=' ')
6 15

A cool feature of Python for loops is tuple assignment. If you are iterating over a list of tuples, the target (word and index were used as the target in the previous two examples) can actually be a tuple of targets. You will see this syntax used a lot with enumerate, and when using dictionary.items().

>>> me = {'name': 'Caiden', 'age': 19}
>>> for key, value in me.items():
...     print(key, ' --- ', value)
name --- Caiden
age --- 19

This is a common scheme in Python. We can print the dictionary key and its value with one for loop. Remember that calling .items() on a dictionary returns a view object which can be treated like a list. The .items() method will return a view object of tuples of a dictionary's key value pairs – meaning that we can iterate over the key value pairs with a for loop using tuple assignment.

Loop Nesting

As you've probably guessed, loop nesting is where you put a loop inside of another loop. Loop nesting is very common, especially in algorithms. Nested loops are very useful when you have something like a 2D or 3D list or tuple. This is where you have a list of lists or a tuple of tuples. That would look something like this [[1, 2, 3], [4, 5, 6]]. As we see here, the objects inside of the list are lists. What if we wanted to find the sum of all values stored in all lists? We cannot do this the same way that we did in the previous list summing example. This would give us an error because the total variable type would not match the object type that you are trying to add to it. In a case like this, we could use a nested for loop.

>>> array_2D = [[1, 2, 3], [4, 5, 6]]
>>> total = 0
>>> for array in array_2D:
...     for number in array:
...         total += number
>>> total
21

In the example, we create a list of lists called array_2D. We then initialize a variable, total, to equal zero. Next, we enter the first for loop. This loop will execute for every object in array_2D. On the next line, we enter the second for loop. This loop will execute for every object in the array object in array_2D. The second for loop will add each number in the current list to the total variable. After every number in the current list has been added to the total, the code will then execute on the next object in array_2D. At the end of the program, when we view the contents of total, we can see that it is the sum of every number in every list in array_2D.
