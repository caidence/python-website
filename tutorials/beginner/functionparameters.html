Parameters aka Arguments

Often times, we write functions to take input from the program, manipulate or use the input in some way, and then return the data after we have used or manipulated the input. We can input information to a function by using what are called arguments, also known as parameters. Look at this statement: def example(a, b):. In the preceding statement, in the parentheses there are two arguments, a and b. These are values that we pass to the function when we call it. To see just exactly what I mean, take a look at the example.

>>> def add_two_numbers(x, y):
...     return x + y
>>> add_two_numbers(5, 7)
12

The function add_two_numbers takes two arguments, x and y. The function will then return the value of the arguments passed to it. Notice how when we call the function, we put the values 5 and 7 in the parentheses. When the function executes from the function call, x will be replaced by 5 and y will be replaced by 7. These are called positional arguments, because the argument variables are determined by the order (position) that we place them in when we call the function. You can also pass arguments by keyword. We can do this by setting the argument that we want to pass in the function call equal to the parameter that we want the argument to be passed as in the function definition.

Keyword Arguments

>>> def display(a, b):
...     print(a)
...     print(b)
>>> display(1, 2)
1
2
>>> display(b=1, a=2)
2
1

We create a function that takes two arguments, a and b. This function simply prints the contents of a and then on a new line, will print the contents of b. The firt time we call the function, we use positional arguments. Because we pass 1 as the first argument, a will take on the value of 1 because a is at the first position in the function definition. b will take on the value of 2 because 2 is the value at the second position in the function call and b is at the second position in the function definition. The second time we call the function, we use what is known as keyword arguments. This is where we use the argument names in the function definition as keywords to pass arguments to the function call. As you can see, we set b=1 and a=2 in the function call. Doing this will set b equal to 1 in the function definition and a equal to 2 in the function definition, because we explicitly tell Python to do so in the function call. Also note, that we put b=2 in the first position. This is just to demonstrate that the position of arguments in the function call is arbitrary when you use keyword arguments.

Defalut Arguments

Python will throw an error if we do not pass any arguments to a function that uses them. To mitigate this, we can use what are known as default arguments. Default arguments are defined in the function definition and are only used if no argument is passed to that specific parameter.

>>> def display(a=0, b=0):
...     print(a, b)
>>> display(1, 2)
1 2
>>> display()
0 0
>>> display(b=7)
0 7

This function is almost the same as the one in the last example. The main difference is: this function has default arguments. Both a and b in this function will default to 0 if no value is passed to them when the function is called. The first time we call display, we pass positional arguments to the parameters a and b. This will print the values that we passed to the function, just like the function in the last example. Next, we call the function without passing any arguments. As you can see, the default values of a and b will then be passed to the function which, of course, prints 0 0. Finally, the third time that we call the function, we only pass a value to b as a keyword argument. Because we did not pass a value to a, it will take on its default value of 0 and the function will print 0 7.
