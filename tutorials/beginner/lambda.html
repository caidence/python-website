The Basics

Python provides a way to create expressions that generate function objects. We can use lambda to create a function to be called later, but creating functions like this will return the function instead of assigning the function to a name. Lambdas are also known as anonymous functions because they are often used inline and do not have a name. If you are familiar with JavaScript, lambda is basically Python's equivalent of an arrow function.

The syntax of this expression is the keyword lambda, followed by one or more arguments, followed by an expression proceeding a colon. There are a few things to keep in-mind when using lambda expressions. Firstly, lambda is an expression that returns a new function which optionally can be assigned a name. With traditional functions, we assign the function a name within the def header rather than returning it as a result. Secondly, a lambda's body is a single, usually succinct, expression; not a body of statements. Lambda expressions are usually short. They were never meant to be a replacement for functions, their purpose is to return a function in places where we cannot use a normal function and/or to package logic into a single line to perform some mundane task that it would be hardly appropriate to create a function for.

Examples and Use cases

The syntax for a lambda expression is as follows lambda arg1, arg2: expression. We create a lambda expression with the lambda keyword, followed by arguments, and finally the expression. Like I stated earlier, we can assign a variable to a lambda expression essentially making the variable a callable function. We could write function = lambda arg1, arg2: expression. In this example, function is assigned the function object that lambda creates.

>>> function = lambda x, y: x ** y
>>> function(2, 3)
8

This example demonstrates how lambda works along with the syntax. We create a variable called function and assign it to the function object created by the lambda expression. In the lambda expression, we declare x and y as arguments before the colon and we return x to the power of y after the colon. We then can use the expression the same way that we call a function by using the variable assigned to the lambda expression essentially as the function name. Lambdas are most commonly used where we need to place small pieces of executable code where statements are illegal.

>>> # Sort items by the second item in the tuple
>>> a = [(1, 3), (4, 1), (7, -1)]
>>> sorted(a, key=lambda x: x[1])
[(7, -1), (4, 1), (1, 3)]
>>>
>>> # Sort items in reverse by the second item in the tuple
>>> sorted(a, key=lambda x: x[1], reverse=True)
[(1, 3), (4, 1), (7, -1)]

The above examples are much more practical use cases for lambda. Python's sorted function accepts an optional keyword argument called key. This argument is a function that serves as a key for the sort comparison. In this case, we use the lambda expression "lambda x: x[1]" which will sort the tuples in the list by the item at the first index. The second example demonstrates how to sort a list in reverse order by the second item in the tuples within the parent list. We can achieve the same result with dictionaries using almost the same syntax. By default, if we do not pass a value to the key argument in sorted with a dictionary, the function will sort the dictionary by the dictionary's keys. We can use this lambda syntax that we just learned to sort a dictionary by its values.

>>> test = {'a': 3, 'b': 1, 'c': 2}
>>> sorted(test.items())
[('a', 3), ('b', 1), ('c', 2)]
>>>
>>> # Sort a dictionary by values
>>> sorted(test.items(), key=lambda x: x[1])
[('b', 1), ('c', 2), ('a', 3)]
>>>
>>> # Sort a dictionary by values in reverse
>>> sorted(test.items(), key=lambda x: x[1], reverse=True)
[('a', 3), ('c', 2), ('b', 1)]

First, we create a dictionary called test with three key value pairs. On the second line, we sort the dictionary with the sorted function. This will sort the dictionary key value pairs by key by default. In the next example, we use the lambda syntax from earlier to sort the dictionary by value. This works because the .items() dictionary method returns a list of two value tuples containing all of a dictionary's key value pairs. Finally, we see how we can sort a dictionary in reverse order by value.

Lambda in lists and dicts

You can branch logic within lists and dictionaries using lambda expressions. To get this functionality, we can use a list/dictionary with lambda expressions packed inside of it the same way as we usually would, but put the parentheses after the statement defining which lambda to use within the list or dict object.

1. power = [
2.     lambda x: x ** 2,
3.     lambda x: x ** 3,
4.     lambda x: x ** 4
5. ]
6. print(power[0](5), end=' ')
7. print(power[1](5), end=' ')
8. print(power[2](5))
25 125 625

In this example, we create a list called power, whose elements are lambda expressions. We can index this list appended with a function call to the lambda expression. "power[0](5)" will call the first lambda expression because it is at index 0 in the list and pass 5 as the argument to lambda. Likewise, "power[1](5)" will retrieve the second index in the power list and pass 5 as an argument to that lambda expression. You can pack as many lambda expressions into a list as you want. This is an easy way to package many simple logic switches together in a single object. We can make this particular program a little more obvious if we use a dictionary rather than a list.

1. power = {
2.     'squared': lambda x: x ** 2,
3.     'cubed': lambda x: x ** 3,
4.     'quartic': lambda x: x ** 4,
5.     'quintic': lambda x: x ** 5
6. }
7. print(power['quintic'](2), end=' ')
8. print(power['cubed'](7))
32 343

This example is practically the same as the last, the difference is we are essentially naming our lambda functions with dictionary keys. We can access the function by fetching the value (lambda function) the same way that we fetch values from any other dictionary. Again, we append the vale getting statement with parentheses, which acts as a function call to the lambda that resides at that particular key. For instance, "power['squared'](3)" will retrieve the value stored at the key "squared" in the dictionary. In this case, the value stored at "squared" is a lambda function. Hence, this statement will return the value of 3 after pushing it through the lambda function stored at the dictionary key "squared".
