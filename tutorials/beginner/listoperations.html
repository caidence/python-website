count, index

Lists have some of the same methods as strings, such as count and index, so we will start with those (these two methods work with all sequence objects). Like with strings, index will return the index of the passed item. Also like with strings, you can pass a second argument telling the index method at what index to start its search at. The count method will return how many times an object appears in a list.

>>> array = [1, 2, 3, 1]
>>> array.count(1)
2
>>> array.index(1)
0
>>> array.index(1, 1)
3

Here we see that count will return the number of times that an element appears in the list. Next, we use index to return the index of the first instance of 1 in the list. The integer 1 appears in the list twice, so count will return 2. Moving on, we see that if you pass a second argument to the index method, it will start the search at that index, just as it will with string objects.

fun with index

This subsection is optional reading, below is a simple program using nested index method calls to print the first and second index of an item input by the user in a list. Line three works by using the index method call we learned earlier, line four is where things get interesting. There is a lot of code on line four, so let's break it down. First, we print "Second index value: ". Then we call the index method on array a first time with the first argument being the index of the value we want to find. The second argument to this call is another call to index again with the same value we want to find, but in this call, we add one to it and pass it as the second argument to the first call to index. Remember that if we pass a second argument to index, Python will start the search at this index in the list. Because the second argument in this case is the index of the first value, we are adding one to that index; therefore, starting the search after the index of the first element.

1. array = [1, 2, 3, 1, 2, 3]
2. value = int(input("Value: "))
3. print("First index of value: ", array.index(value))
4. print("Second index of value: ", array.index(value, array.index(value) + 1))

append, insert, pop, remove

You will probably use append a lot in your programs. Append takes one argument and adds that argument to the end of a list. Insert accepts two arguments, the first being an index of where to put an item at in a list, and the second being the item to put at the index passed as the first argument. pop, by default, will remove the last element in a list and return its value. Pop accepts an optional argument, an index. If you pass this argument to pop, it will remove that index from the list and return its value. It is common to store pop in a variable so that you can modify it or use it later. Remove accepts an object as an argument and removes that object from the list.

>>> # Append example
>>> array = [1, 2, 3]
>>> array.append(4)
>>> array
[1, 2, 3, 4]

We can see here that all the append method really does is add an item to the end of the list. In the example, we append 4 to the end of the list object. You can also append items to a list by using the += operator, essentially "adding" an item to the end of a list.

Insert accepts two arguments, the first being an index, and the second being an object. The first argument (the index) is used to place the second argument (the object) at that index in the list. For example, if we have a list array and we want to place the number 10 at the first index of the list, we could write array.insert(0, 10). The first argument is the index to place the number 10 (the second argument) at in the list. This will not replace the object at the first index, it will just shift it along with all of the other objects in the list to the right.

>>> # Insert example
>>> array = [2, 3]
>>> array.insert(0, 1)
>>> array
[1, 2, 3]

In this example, we insert the number 1 at the 0th index of the list. Remember that list indices start counting at 0, not 1. The second argument passed is 1, meaning that we want to place 1 at the 0th index of the list, prepending the list with the value 1.

pop will remove the last item of the list and return it. Pop accepts an optional argument, though. As previously discussed, the optional argument that you can pass to pop is the index of the item in the list you would like to remove or, "pop off the list".

>>> # pop examples
>>> array = [1, 2, 3]
>>> array.pop()
3
>>> array
[1, 2]
>>> array = [1, 2, 3]
>>> array.pop(0)
1
>>> array
[2, 3]

In this example, we start with an array [1, 2, 3]. Next, we use the pop method on array with no arguments. This removes the last element of the list and returns it. As you can see, after we call the pop method on array, it has removed the last element. Next, we are just resetting the value of array to [1, 2, 3]. After that, we call the pop method on array passing an argument, the index of the item we would like to pop off of the list. In this example, we pop the first item off of the list by passing the index 0 to the pop method. Now if we view the contents of array, we see that pop has removed the first item from the list.

The remove method removes an object from a list. It accepts one argument, the object to remove. Say we have a list called example. Its value is ['a', 'b', 'd', 'c']. We want to remove the character 'd' from this list, but how? We could do something like this example.remove('d'). This will remove the string object 'd' from our list.

>>> # remove example
>>> example = ['a', 'b', 'd', 'c']
>>> example.remove('d')
>>> example
['a', 'b', 'c']

We start with ['a', 'b', 'd', 'c']. We want to remove the string object 'd' from our list, so we simply pass it as an argument to the remove method like this example.remove('d'). Remove is a simple method: you just call it on the list you want to remove something from, with the object that you want to remove from the list as an argument.

sort, reverse

I'm sure that you can guess what these two methods do. sort will sort a list in ascending order, and reverse will reverse all of the elements in a list, making a mirror image of the list. If you want to sort a list in reverse order, you could call the reverse method on a list that has already had the sort method called on it, but there is another cool way to do this, only using the sort method.

>>> # Sort in ascending order
>>> array = [2, 1, 4, 3]
>>> array.sort()
>>> array
[1, 2, 3, 4]
>>> # Sort in descending order
>>> array.sort(reverse=True)
[4, 3, 2, 1]

Next, we are going to talk about sort and reverse. These are not the only ways to sort and reverse sequences in Python, but they are two of the most common. First, we create an array [2, 1, 4, 3]. Next, we call the sort method on it. This, by default, will sort the array in ascending order. We can see how to sort the list in descending order on the following lines. We can do this by adding an argument switch to the sort method, reverse=True. When you have an argument and set it equal to a Boolean value, this is often called a switch, because you are either switching on or off some functionality of a method or function.

>>> example = [2, 1, 3]
>>> example.reverse()
>>> example
[3, 1, 2]
>>> example2 = ['a', 'c', 'b']
>>> example2.reverse()
['b', 'c', 'a']

In the example, we create a list called example and set its value as [2, 1, 3]. Next, we call the reverse method on the list. Now when we view the contents of the list; it will be in reverse order of what it previously was. Note, calling reverse on a list does not sort the list in reverse order, it simply reverses the current order of the objects in the list. Continuing with our example, we can see that reverse also works on a list of strings.
