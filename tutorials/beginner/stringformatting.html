String Formatting

In short, string formatting provides us a simple way to create basic template strings where we can pass values into that template to change the value of the final string. There are many ways to format a string, but we will only be looking at a few of the most common in this tutorial. With string formatting, we can perform multiple substitutions on a string in a single expression.

Formatting Expressions

A very common way to format a string is by using the % character. Using %, we can code many substrings all at once into one string template instead of using longer concatenation expressions. On the left side of %, we put a string template with one or more targets of where we want to format data to. On the right side of %, we put the objects embedded within a tuple, that we want to format into the string template. The syntax is "--- %s --- %s ---" % (target 1, target 2).

>>> # Type specific
>>> "%d good %s 4 you" % (1, "example")
1 good example 4 you
>>> # All types match %s
>>> "%s, %s, %s" % (7.9, [8, 2, 1], 3)
7.9, [8, 2, 1], 3

In the first example, the %d in the string is replaced with a 1 and %s is replaced with "example". %s works with all data types in string formatting. It is unlikely that you will use anything but %s. %s works with every type, so unless you'll be doing specific string formatting, %s is all you need to use in formatting expressions. There are other types besides %s and %d, here are some of the most common.

1. %s # String                  5. %c # Character (int or string)
2, %r # repr                    6. %o, %O # Octal, base 8
3. %d # Decimal, base 10        7. %x, %X # Hex Integer, Base 16
4. %i # Integer                 8. %e, %E, %f, %F, %g, %G # Floating point number

The Format Call

The format method is available in all releases of Python 3. It is fairly straight forward, you just call the method on a template string with arguments as the target values. template.format(target 1, target 2) is the syntax when using the format method. template can either be a string or a variable holding a string template, .format is the format method, and target 1 and target 2 are the values that will be formatted into the string template. You can pass arguments to the format method by keyword, position, or both. In the template string, we put curly braces { } where we want the arguments from the format call to appear in the string template.

>>> # Positional arguments
>>> "{0}, {1}, and {2}".format(1, 2, 3)
1, 2, and 3
>>> # Keyword arguments
>>> "{one}, {two}, and {three}".format(three=3, two=2, one=1)
1, 2, and 3
>>> # Keyword and positional arguments
>>> "{one}, {0}, and {three}".format(2, one=1, three=3)
1, 2, and 3
>>> # Positional arguments
>>> "{}, {}, and {}".format(1, 2, 3)
1, 2, and 3

In the first example, we pass the items to the string based on their index in the argument list to the format call. Next, we pass the values to the string by keyword. The way that you pass a value by keyword to the function call is by placing a keyword in curly braces within the string template and then set their value equal to the method call to format. In the next example, we see that you can use both positional and keyword arguments in the same method call. The final example is the same as the first. The reason that we don't use numbers in the curly braces is because positional arguments are default with the format call; meaning, the numbers are implied in the last example.

You can also use lists to format strings with the format method call. In the string template, within the curly braces, we can pass the index of the list object in the argument list to format and then we can use list indices on that list object in the template.

>>> pets = ["cat", "dog", "fish"]
>>> pet_names = ["Kitty", "Puggles", "Ball"]
>>> print("I have a {0[0]} named {1[0]}.".format(pets, pet_names))
I have a cat named Kitty.
>>> print("I have a {0[2]} named {1[2]}.".format(pets, pet_names))
I have a fish named Ball.

In the examples, we pass two lists as arguments to the format method. In the curly braces, the first number is selecting which list to use. This number represents the index of the object in the argument list to use. If the first number is 0, it will select the first list. Likewise, if the first number is 1, the object selected will be the object at the second index in the argument list. The second number (the number in square braces [ ]) proceeding the first number in the string template is the index of the object to format into the string template. For instance, {0[0]} means to select the first list in the format method call and format the 0th index of that list into the string template.

Unpacking Dictionaries

We will cover this in more depth in a later tutorial, so do not worry if you do not understand the syntax used in this subsection. Python has a feature called unpacking. This is commonly used with lists and dictionaries, to unpack their values either to a function or, in this case, a string template. We can do this by putting a dictionary's key in the curly braces within the string template. Then when we use the unpacking syntax on the template with the dictionary, the values associated with the keys in the curly braces will be formatted into the string.

>>> caid = {"first_name": "Caiden", "last_name": "Pyle"}
>>> template = "Hello {first_name} {last_name}"
>>> print(template.format(**caid))
Hello Caiden Pyle

First, we create a dictionary named caid and set the key first_name equal to "Caiden" and the key last_name equal to "Pyle". Next, we create a string template with the dictionary's key names inside of the template's curly braces. The key names must be exactly the same within the template as they are in the dictionary. Now when we unpack the dictionary into the string template, the values associated with the keys in the curly braces within the string template will be formatted into the string. In the format method call, notice the double asterisk before the dictionary name. This is how we unpack the dictionary caid into the string template template.

F" "

There is a way that we can implant variable names directly into a string template to where when the string is printed the variable names will be replaced with the value held in that variable. This is basically the same thing as templating in JavaScript.

name = input("Name: ")
age = input("Age: ")
print(f"{name} is {age} years old.")

The syntax is the same as raw strings. Place an "f" right before the opening quotation mark of the string and place the variable name(s) directly in the curly braces of the string template. This is a very helpful feature, but it is not commonly used in everyday practice. At the time of this writing, using f" " to format strings is not very common since it is not supported in all versions of Python.
