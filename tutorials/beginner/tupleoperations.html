count, index

Tuples are immutable objects in Python. As you can imagine, this greatly reduces how many methods there are for this object type. The two that we will talk about in this section are count and index. They do exactly the same things as the list count and index methods. Here are some examples to demonstrate.

>>> sequence = (1, 2, 3, 1, 2, 3)
>>> sequence.count(1)
2
>>> sequence.index(1)
0
>>> sequence.index(1, 1)
3

Here we create a tuple named sequence. We call the count method on it passing a single argument, 1. Like with lists, the count method will return how many times the argument passed appears in the tuple object. Because the integer 1 appears twice in the tuple sequence, the count method returns 2. Moving on to index. Index also works just as it does with lists. If you pass a single argument to the index method, it will return the index of the first instance of the object that it finds in the tuple. If you pass a second argument to index, Python will start the search from this index on.
