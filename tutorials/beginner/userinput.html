The Basics

You've learned how to output data to the console using the print function; similarly, you can input data from the console using the input function. All you have to do is write input( ). Then the program will stop, wait for the user to input something, and then continue once the user presses enter. More often than not, you would store this input in a variable and use it later in your code. You can also display a message with the input function – just add a string containing your message inside of the braces of the input function. This string usually directs to the user on what the program needs the them to input.

>>> # Input examples
>>> input()
>>> variable = input()
>>> name = input("Enter your name: ")

In the example, the first line will wait for the user to input something and then the program will continue to execute after the user presses enter. The second line is the same as the first, but it will store what the user inputs in a variable so that we can use it later in our code. The third line will prompt the user "Enter your name: ", and store whatever the user enters in a variable called "name".

Examples

Like I said earlier, more often than not, input is stored in a variable so that we can use it later in the program. Here is a simple example of a program that will ask a user for their name and tell them hello.

>>> name = input("What's your name? ")
>>> print("Hello " + name)

This program will prompt the user, asking them to enter their name. After the user inputs their name, what was entered is stored in a variable called name so that we can use it later in our program. Finally, the program will print the string: "Hello " and the user's name that was stored in the variable name. Pretty cool stuff, right?

The input function will turn everything that you input into a string by default. This makes it impossible to do math with anything that the user inputs without using a neat trick that I will demonstrate next.

Type conversion

In many other programming languages, this is called type casting. Type conversion is where you take one data type and turn it into another. In a previous tutorial, you learned about data types. The data types we will be focusing on in this section are string (str), integer (int), and floating point numbers (float); for the sake of simplicity.

>>> # Converting float to int
>>> int(1.8)
1
>>> # Converting int to float
>>> float(7)
7.0
>>> # int to string
>>> str(7)
'7'

Converting between data types is simple, you just take the data that you want to convert, and wrap it with the data type you want it to be. When I talk about "wrapping", I really mean something like this int(data to convert). This example will convert whatever type the data is in the parentheses to int data type. type conversion and input

The reason that I bring up type conversions in this section is because they are very common to use with the input function. Like I said earlier, what if you want the user to input a number so that you can do math with it? Python doesn't allow you to do math with a string, you would need an int or a float. Type conversions make it fairly easy to solve this problem.

1. # You could do this
2. age = input("Enter your age: ")
3. int_age = int(age)
4.
5. # This is more common
6. age = int(input("Enter your age: "))

On line two, the input that the user enters will be a string by default. This is why on line three we convert the age variable (which is a string) to an int. Line six is more common and does the same thing as lines two and three combined. Since we are converting what the user types from a string into a integer, now we can do math with it! You can convert the input variable to an int after it is assigned, but it is more common to cast it to the data type you want on the same line as the input function. Lines two and three in the example do the exact thing as line six, it is up to the programmer to decide how they want to convert input to another type.
