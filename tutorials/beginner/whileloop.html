While Loop

The while loop is simple to understand. Basically, it repeatedly executes a block of code until a condition evaluates to false. After all the code in the loop runs, the condition is evaluated again, if the condition is still true, the loop will run again. Usually, within the while loop there is code to update the condition, so that the loop will eventually evaluate to false and end.

>>> name = "Caiden"
>>> while name:
...     print(name, end= ' ')
...     name = name[1:]
Caiden aiden iden den en n

This program will print the string as long as the while loop evaluates to true. Remember, any string containing any characters evaluates to true in Python. After name is printed, the name variable is changed to equal itself with the first character removed. The while loop then executes again, printing the new name variable, updating the name variable by removing the first character again, and keeps executing until there are no characters left in the name variable.

While loops are a good place to ask for user input. They are great for making a menu of options for your programs when you want an interactive interface. You can easily write a program that continues to print a string until a certain character is input.

>>> choice = None
>>> while choice != "quit":
...     choice = input("Type 'quit' to exit the loop.").lower()

This is not a very practical program, but it demonstrates a point. On the first line, we set choice equal to None. This is often a placeholder keyword used just to initialize variables, but not give them a value yet. Next, we start the loop which will not end until the user inputs "quit". This loop will keep prompting the user "Type 'quit' to exit the loop." until the user types "quit". Because we call the str.lower() method on the input statement, the user can type "Quit", "QUIT" or "quit", and the loop will still end.

>>> condition = 0
>>> while condition < 5:
...     print(condition, end=' ')
...     condition += 1
0 1 2 3 4

This program prints the values 0 through 4. It starts by evaluating the variable condition. If the condition is less than 5, the loop will print its value and then add one to the it. This will continue to happen until eventually, the condition is met and the statement evaluates to false, which ends the loop.
