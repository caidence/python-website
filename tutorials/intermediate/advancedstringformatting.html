The Basics

String formatting is usually used to present data in a certain way. The syntax can be intimidating and pretty complex. It definitely takes some getting used to, but it is very powerful and well worth learning for sure! You can do almost anything you want with the syntax that you're about to learn. We'll start with string alignment using the format call.

Alignment

>>> "{0:10} - {0:>10}".format("spam")
'spam      =      spam'

In the string template, inside of the curly braces, the first number is the index of the argument in the format call. {:10} means to put the first positional argument in a field that is 10 characters wide. {0:>10} means to put the first positional argument into a string that is 10 characters wide where the text is aligned to the right. If the text is more than 10 characters, you will not notice a difference in the alignment.

1. print("{:>25}".format("I'm aligned right"))
2. print("{:<25}".format("I'm aligned left"))
3. print("{:^25".format("I'm centered"))
"        I'm aligned right"
"I'm aligned left         "
"      I'm centered       "

The first line puts the text into a string that is 25-characters wide and aligns the text to the right. The second line puts the text into a 25-character long string and aligns the text to the left. The final example puts the text into a 25-character long string and aligns the text in the center of it. The "padding" character is a space by default, but we can change this by placing a character before the alignment character (<, >, ^) in the string formatting template.

1. print("{:!>25}".format("I'm aligned right"))
2. print("{:@<25}".format("I'm aligned left"))
3. print("{:=^25}".format("I'm centered"))
"!!!!!!!!I'm aligned right"
"I'm aligned left@@@@@@@@@"
"======I'm centered======="

Working with Numbers

You can also convert to other types using string formatting. For instance, you can convert integers to binary numbers by using a simple string format call.

1. number = int(input("Number to convert: "))
2. print("Binary: {:b}".format(number))

This program asks the user for a number to convert to binary and then prints that number in binary format. It begins by storing the integer value input by the user in a variable called number. The second line of code takes the value stored in number, converts it to binary, and prints it in string format. Notice how we cast the input to be of type integer. This is because you cannot convert a float to binary.

String formatting also makes it easy to convert decimal numbers to a percent value. We can do this by using the syntax: "{:%}".format(decimal).

>>> questions = 93
>>> correct = 81
>>> "Grade: {:%}".format(correct/questions)
'Grade: 87.096774%'
>>> "Grade: {:.2%}".format(correct/questions)
'Grade: 87.10%'
>>> "Grade: {:.3%}".format(correct/questions)
'Grade: 87.097%'

This is a simple way to print percentages in a more easy-to-read format. In the curly braces within the template string, the % tells Python that we want to format a decimal number to a percentage. {:.2%} means that we want to format a decimal to percent format and to print only two decimal places. Likewise, {:.3%} will format a decimal number to a percentage with three decimal places. This is also a good example demonstrating that we can put more than one formatting instruction within a single pair of curly braces.

String formatting also makes it relatively easy to format numbers in order to make them more readable. For instance, we can easily comma separate large numbers and choose how many decimal places that we want. Have a look at the examples for clarification.

>>> print("{:,}".format(1000000))
1,000,000
>>> print("{:.2f}".format(100))
100.00
>>> print("{:,.4f}".format(10000))
10,000.0000

The first example is how to comma separate large numbers. Comma separate means to place a comma every three digits going from right to left. The second example demonstrates how to get a specific number of digits after the decimal place. "{:.2f}" means to format the number to have two digits to the right of the decimal. The "f" in the expression "{:.2f}" specifies that we want this to be formatted to look like a floating point number.
