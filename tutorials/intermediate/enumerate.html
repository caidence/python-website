The Basics

Enumerate is a very useful built in Python function. It essentially "counts", the values in an iterable object. I use the word "count" loosely because this is not actually what the function does, this is just a common use case. enumerate(iterable) will return an object that is basically a list of two value tuples. The first value in the tuples is a number and the second value is a single index of the iterable object.

>>> enumerate('test')
<enumerate object at 0x0000015F46A85C18>
>>> list(enumerate('test'))
[(0, 't'), (1, 'e'), (2, 's'), (3, 't')]
>>> list(enumerate('test', 5))
[(5, 't'), (6, 'e'), (7, 's'), (8, 't')]

As you can see, the enumerate function will return a new object. If you wish to view the contents of this object, you need to wrap it with list( ). Have a look at the output. We see that the object created by enumerate is a list of two item tuples. The first item in the tuple is the counter. This is the number at which the counting is started. By default this value convinently is 0. The second item in the tuple is an index of the iterable. We can see that the counter item increments by one with every index in the iterable object. We can pass a second optional argument to enumerate, a value at which we want the counter to start at. If we pass this argument to enumerate, it will start counting at that value. For instance, in the example in the last call to enumerate, we pass 5 as the second argument. In the output we see that the first value of the first tuple is 5.

Using Enumerate

There are more use cases for enumerate that I will demonstrate in later tutorials, these examples are just a few common use cases. We can use enumerate to easily number lists and pick out certain indices of iterable objects. For example, this program numbers a shopping list.

>>> shopping_list = ['apples', 'eggs', 'milk']
>>> for counter, item in enumerate(shopping_list, 1):
...     print(f'{counter}. {item}')
1. apples
2. eggs
3. milk

On the first line of code, we create a variable called shopping_list which contains a list of items that you can buy at a grocery store. The next line starts a for loop. The loop variables can be confusing, so let me explain them. counter is referring to the first value in each tuple within the enumerate object. Remember, this is the value that is incremented by one with each progressing tuple. In this example, the loop variable item is actually referring to the item in the iterable object or the second value in each tuple in the enumerate object. The name of these loop variables is arbitrary, it is the position of these variables in the loop header that matters. Next, we just print a formatted string with the counter value on the left and the item's value on the right.

>>> name = "Tommy Le"
>>> upper_indices = []
>>> for index, char in enumerate(name):
...     if char.isupper():
...         upper_indices.append(index)
>>> upper_indices
[0, 6]

The first two lines of the example above are pretty straight forward. On the third line, we begin the for loop. Within the for loop, each element in the iterable object is checked to see if it is an uppercase letter. If the item is an uppercase letter, the index of that item in the object is appended to the list upper_indices. This works because enumerate starts counting at 0 by default. Conveniently, indices of items in iterable objects also start counting at 0 by default. This means that the enumerate counter will match an item's index in an iterable object. The final line of code in this example is to view the contents of upper_indices. We can see that our program worked and returns the indices of the uppercase letters in the string name.
